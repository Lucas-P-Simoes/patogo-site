Como instalar a extensão PatoGo no Chrome (unpacked)

  1. Baixe o ZIP da versão: PatoGo 1.0.0 .
  2. Descompacte para uma pasta fixa, ex.: C:\\PatoGo\\Extension.
  3. Abra chrome://extensions → ative Developer mode (canto superior direito).
  4. Clique em Load unpacked e selecione a pasta onde você extraiu.
  5. Fixe o ícone do PatoGo na barra para acesso rápido.

Como atualizar

  1. Quando a extensão avisar, volte aqui e baixe o novo pacote.
  2. Substitua os arquivos na pasta de instalação (ex.: C:\\PatoGo\\Extension).
  3. Abra chrome://extensions e clique em Update (ou Reload na extensão).

FAQ — Perguntas frequentes

A PatoGo instala pela Chrome Web Store?
Não. O patogo foi feito para ser instalado pelo modo desenvolvedor, assim evitando bloqueios.
A PatoGo verifica se o número tem WhatsApp?
Sim. A extensão abre o WhatsApp Web com o número informado e retorna o status (válido ou não) diretamente no navegador.
Recebo atualização automática?
A extensão avisa quando há nova versão e você atualiza seguindo os passos.
